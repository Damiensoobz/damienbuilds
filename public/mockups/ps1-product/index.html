3D Pioneer

The PlayStation was the first console to make real-time 3D gameplay mainstream — changing game design forever.

CD-ROM Media

Ditched the expensive cartridge format for CD-ROMs — giving developers freedom to create deeper, richer worlds with full soundtracks.
